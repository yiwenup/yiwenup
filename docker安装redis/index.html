03_Docker 安装 Redis

 收录于 Docker
 2021-11-12  约 280 字   预计阅读 1 分钟 
目录

  1. 获取镜像

    1

    docker pull redis:6.2.6

    1

    docker images

  2. 创建本地挂载目录

    1

    mkdir -p ~/docker/redis

  3. 预创建本地映射的redis.conf文件，否则待容器启动后，redis.conf将被误处理为文件夹

    1

    mkdir -p ~/docker/redis/conf/

    1

    cd ~/docker/redis/conf/

    1

    touch redis.conf

  4. 创建容器

    1
2
3
4
5
6

    docker run -p 6379:6379 --name redis \
-v ~/docker/redis/data:/data \
-v ~/docker/redis/conf/redis.conf:/etc/redis/redis.conf \
--restart=always \
--privileged=true \
-d redis:6.2.6 redis-server /etc/redis/redis.conf

  5. 修改redis.conf

    全量配置项见官网：https://raw.githubusercontent.com/redis/redis/6.2/redis.conf

    • requirepass：Redis 访问密码
    • appendonly：AOF 持久化
    • bind：修改为bind 0.0.0.0，否则远程无法访问

  6. 重启容器，查看是否重启成功

    1

    docker restart redis

    1

    docker ps
