03 Zookeeper基本命令

 收录于 Zookeeper
 2024-03-02  约 255 字   预计阅读 1 分钟 
目录

创建节点

1
2

# 临时节点无法创建子节点
create [-s] [-e] [-c] [-t ttl] path [data] [acl]
  • -s 表示创建顺序节点
  • -e 表示创建临时节点
  • -c 表示创建容器节点

查看节点

1

ls [-s] [-w] [-R] path
  • -s：列出当前节点的子节点和当前节点的信息
  • -w：仅列出当前节点的子节点**（会注册监听，监听是一次性的）**
  • -R：列出当前节点的所有子节点（递归）
1

get [-s] [-w] path
  • -s：获取当前节点的数据和节点信息
  • -w：仅获取当前节点的信息**（会注册监听，监听是一次性的）**

修改节点数据

1

set [-s] [-v version] path data
  • -s：修改当前节点数据，并展示节点信息

删除节点

1
2

# 无法删除有子节点的节点，如需批量删除可用 deleteall
delete [-v version] path
